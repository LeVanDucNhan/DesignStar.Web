Tải DesignStar

DesignStar dùng 1 file cài đặt. Bạn có thể dùng thử 15 ngày, sau khi mua chỉ cần nhập License Key để kích hoạt Full (không cần tải file Full riêng).

Gợi ý: Nếu bạn đã mua bản quyền, hãy cài DesignStar như bình thường, sau đó vào mục Kích hoạt trong phần mềm để nhập License Key.

Yêu cầu hệ thống

  • Windows 10/11 (64-bit)
  • .NET Desktop Runtime (nếu phần mềm yêu cầu)
  • Quyền chạy file cài đặt (khuyến nghị Run as Administrator nếu bị chặn)

Cách cài đặt (từng bước)

  1. Bấm nút Tải DesignStar ở trên.
  2. Tải file DesignStar.rar → giải nén → chạy DesignStar.exe.
  3. Nếu Windows hiện cảnh báo bảo mật → chọn More info → Run anyway (nếu bạn tin file do bạn phát hành).
  4. Next → Install → Finish.
  5. Mở DesignStar từ Desktop/Start Menu.
  6. Nếu bạn đã mua: vào mục Kích hoạt → nhập License Key.

Nếu báo SmartScreen

Windows SmartScreen thường xuất hiện với phần mềm chưa ký code-signing. Bạn có thể: Click “More info” → “Run anyway”.
