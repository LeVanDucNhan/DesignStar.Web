Tải TowerDraft

Chọn 1 trong 2: Trial (15 ngày) hoặc Full (nếu bạn đã có license key). Bạn sẽ thay link tải bên dưới bằng link GitHub Releases của bạn.

⬇️ Download Trial Installer ⬇️ Download Full Installer
Lưu ý: Nếu bạn muốn đơn giản nhất, bạn có thể chỉ dùng 1 file installer (Trial), còn sau khi mua thì người dùng chỉ cần nhập key để thành Full (không cần tải file Full).

Yêu cầu hệ thống

  • Windows 10/11
  • .NET Desktop Runtime (nếu app bạn cần)
  • Quyền chạy installer (khuyến nghị Run as Administrator nếu bị chặn)

Cách cài đặt (siêu chậm)

  1. Bấm nút Download ở trên.
  2. Chờ tải xong → mở file .exe.
  3. Nếu Windows hỏi “More info” → chọn Run anyway (nếu bạn tin file do bạn phát hành).
  4. Next → Install → Finish.
  5. Mở TowerDraft từ Desktop/Start Menu.

Nếu báo SmartScreen

Windows SmartScreen thường xuất hiện với phần mềm chưa ký code-signing. Bạn có thể: Click “More info” → “Run anyway”.
